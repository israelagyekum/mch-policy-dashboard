# Maternal & Child Health Policy Intelligence Platform

## Overview
This project is an AI-powered decision support system designed to assist policymakers in analysing maternal health outcomes using econometric modelling and machine learning.

The platform integrates:
- Two-Way Fixed Effects Panel Regression (Econometrics)
- Random Forest & Gradient Boosting (Machine Learning)
- Explainable AI (SHAP)
- Interactive Policy Simulation Dashboard (Streamlit)

---

## Key Features

### 1. Policy Simulation
Simulate changes in:
- GDP per capita
- Fertility rate
- Health expenditure
- Female secondary education

Observe estimated impact on maternal mortality.

---

### 2. Hybrid Modelling Framework
- Econometric model for causal interpretation
- Machine learning models for predictive benchmarking

---

### 3. Explainable AI
- SHAP-based feature importance
- Model transparency for decision support

---

### 4. Executive Dashboard
- Interactive filters (Country, Year)
- KPI tracking
- Policy impact visualisation
- Model diagnostics

---

## Data Source
World Bank Dataset (2000–2022)
